# Cofoundry.Plugins.YouTube

This library is a plugin for [Cofoundry](https://www.cofoundry.org/). For more information on getting started with Cofoundry read the [Cofoundry documentation](https://www.cofoundry.org/docs).

## Overview

This plugin adds a `[YouTube]` data model attribute and UI editor.

For more information read the [YouTube plugin documentation](https://www.cofoundry.org/docs/plugins/available-plugins/youtube).
