# Cofoundry.Plugins.SiteMap

This library is a plugin for [Cofoundry](https://www.cofoundry.org/). For more information on getting started with Cofoundry read the [Cofoundry documentation](https://www.cofoundry.org/docs).

## Overview

The SiteMap plugin is a quick and easy way to add a dynamic sitemap to your site. 

For more information read the [SiteMap plugin documentation](https://www.cofoundry.org/docs/plugins/available-plugins/sitemap).
