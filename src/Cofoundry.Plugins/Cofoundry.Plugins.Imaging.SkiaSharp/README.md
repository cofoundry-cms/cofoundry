# Cofoundry.Plugins.Imaging.SkiaSharp

This library is a plugin for [Cofoundry](https://www.cofoundry.org/). For more information on getting started with Cofoundry read the [Cofoundry documentation](https://www.cofoundry.org/docs).

## Overview

This plugin uses the [SkiaSharp](https://github.com/mono/SkiaSharp) wrapper library to resize images dynamically. 

For more information read the [Imaging.SkiaSharp plugin documentation](https://www.cofoundry.org/docs/plugins/available-plugins/imaging-skiasharp).
