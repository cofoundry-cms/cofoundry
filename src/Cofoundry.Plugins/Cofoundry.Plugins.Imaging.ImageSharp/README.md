# Cofoundry.Plugins.Imaging.ImageSharp

This library is a plugin for [Cofoundry](https://www.cofoundry.org/). For more information on getting started with Cofoundry read the [Cofoundry documentation](https://www.cofoundry.org/docs).

## Overview

This plugin uses the cross platform [ImageSharp](https://github.com/SixLabors/ImageSharp) package to resize images dynamically. 

For more information read the [Imaging.ImageSharp plugin documentation](https://www.cofoundry.org/docs/plugins/available-plugins/imaging-imagesharp).
