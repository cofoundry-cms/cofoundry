# Cofoundry.Plugins.Mail.MailKit

This library is a plugin for [Cofoundry](https://www.cofoundry.org/). For more information on getting started with Cofoundry read the [Cofoundry documentation](https://www.cofoundry.org/docs).

## Overview

This library allows you to send mail using the cross platform [MailKit](https://github.com/jstedfast/MailKit) library. 

For more information read the [Mail.MailKit plugin documentation](https://www.cofoundry.org/docs/plugins/available-plugins/mail-mailkit).
