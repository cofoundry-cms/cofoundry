# Cofoundry.Plugins.Mail.SendGrid

This library is a plugin for [Cofoundry](https://www.cofoundry.org/). For more information on getting started with Cofoundry read the [Cofoundry documentation](https://www.cofoundry.org/docs).

## Overview

TThis plugin allows you to send mail using the [SendGrid](https://sendgrid.com/) api. 

For more information read the [Mail.SendGrid plugin documentation](https://www.cofoundry.org/docs/plugins/available-plugins/mail-sendgrid).
